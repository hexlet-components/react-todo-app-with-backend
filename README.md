# hexlet-todo

Пример реализации ToDo-list на React.

## Участие

* Обсуждение в канале [#hexlet-volunteers](https://slack-ru.hexlet.io) слака

## Системные требования

* Node.js >= 14
* npm >= 7
* make >= 4 (optional)

## Установка

```shell
make setup
make start # running server on localhost
```
